# ALLSHOES - Proyecto Web HTML/CSS

## Descripción General
Este proyecto consiste en la creación de un sitio web. El objetivo principal ha sido convertir cinco mockups de diseño gráfico en páginas web funcionales e interactivas, utilizando exclusivamente HTML y CSS.

## Estructura del Proyecto
El sitio está compuesto por cinco páginas principales interconectadas mediante una cabecera de navegación común:

* **Proyecto.html (Inicio):** Página principal con un banner introductorio y una galería destacada.
* **Sobre_nosotros.html:** Página corporativa estructurada visualmente para acomodar el texto descriptivo y la imagen de la empresa.
* **Cursos.html:** Sección que muestra la oferta formativa mediante un diseño de tarjetas alternas con imágenes y texto.
* **Tienda.html:** Catálogo de productos organizado en cuadrícula, mostrando imágenes, precios y botones de acción, además de un diseño visual de paginación.
* **Contacto.html:** Página de atención al cliente dividida en dos secciones: información de contacto con enlaces a redes sociales, y un formulario interactivo en la columna derecha.

## Detalles Técnicos y Desarrollo
Para cumplir con los requisitos del proyecto, se han aplicado los siguientes criterios:

* **Tecnologías:** Desarrollo realizado íntegramente con HTML5 y CSS3, sin uso de JavaScript.
* **Fidelidad al diseño:** Uso de CSS (especialmente Flexbox) para lograr una disposición exacta de los elementos, colores y tipografías basados en los mockups originales.
* **Interactividad:** Implementación de transiciones suaves y efectos visuales (`hover`) en botones, enlaces e imágenes para mejorar la experiencia de usuario.
* **Convenciones de código:** Código estructurado y comentado solo en las secciones necesarias. Todos los nombres de clases e identificadores (`id`) están escritos en español y describen claramente su función o contenido.

## Instrucciones de Navegación
Para visualizar el proyecto:
1. Extraiga los archivos del documento `.zip` en su equipo.
2. Abra el archivo `index.html` en cualquier navegador web moderno.
3. Utilice el menú de navegación superior para desplazarse entre las diferentes secciones del sitio (Inicio, Sobre Nosotros, Cursos, Tienda y Contacto).
